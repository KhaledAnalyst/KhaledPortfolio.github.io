Microsoft Excel

Excel is a tool that has features like calculation, graphing tools, pivot tables, and a macros programming language.
And being a flexible, easy-to-use spreadsheet, it is often used to create datasets. It can pull data from external sources into your spreadsheet with the help of data connection features. It can also obtain data from sources such as the Web, Microsoft Query, SharePoint List, OData Feed, Hadoop Files (HDFS), etc. Consequently, Excel files are often used as a data source for Power BI and Tableau.
